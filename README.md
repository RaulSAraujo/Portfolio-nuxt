# Portfolio Nuxt

Portfólio pessoal desenvolvido com [Nuxt](https://nuxt.com), [Nuxt UI](https://ui.nuxt.com) e [Nuxt Content](https://content.nuxt.com).

## Stack

- **Nuxt 4** – Framework Vue para aplicações web
- **Nuxt UI** – Componentes e estilos
- **Nuxt Content** – Conteúdo em YAML/Markdown
- **Tailwind CSS** – Estilização
- **Motion (Vue)** – Animações
- **TypeScript** – Tipagem estática

## Pré-requisitos

- [Node.js](https://nodejs.org/) (LTS recomendado)
- [pnpm](https://pnpm.io/) (`npm install -g pnpm`)

## Instalação

```bash
pnpm install
```

## Desenvolvimento

Inicia o servidor de desenvolvimento em `http://localhost:3000`:

```bash
pnpm dev
```

## Build para produção

```bash
pnpm build
```

## Preview da build de produção

```bash
pnpm preview
```

## Outros comandos

- `pnpm lint` – Executa o ESLint
- `pnpm typecheck` – Verifica tipos TypeScript
